# 대전여지도

대전여지도는 Google Maps API를 활용해 대전 지역의 맛집 정보를 지도 위에 시각화하는 토이 프로젝트입니다.

Spring Boot 백엔드에서 맛집의 이름, 주소, 카테고리, 위도, 경도 데이터를 REST API로 제공하고, 
React 프론트엔드에서는 해당 데이터를 비동기로 조회해 Google Maps 지도 위에 마커로 렌더링합니다.

이 프로젝트는 백엔드 개발자 지망생으로서 지도 기반 서비스에서 API 데이터가 프론트엔드 화면에 어떻게 연결되는지 이해하고, 
Google Maps API, RESTful API, React 기초 활용 경험을 직접 쌓기 위해 진행했습니다.
